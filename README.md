## Quick Start

### Prerequisites

- Docker and Docker Compose
- OpenAI API key
- Tavily API key

### Installation

1. Set up your environment variables:
```bash
OPENAI_API_KEY=your_openai_key_here
TAVILY_API_KEY=your_tavily_key_here
```

2. Launch with Docker:
```bash
docker compose up
```
